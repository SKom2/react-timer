# Simple React Timer 

![image](https://github.com/SKom2/react-timer/assets/103752057/cccb9941-4696-41ba-a3ea-bd9d0278d00a)

## Возможности
- Установка таймера
- Установка времени на таймере
- Сброс таймера
- Постановка таймера на паузу

## Описание
Простой таймер на React предоставляет вам возможность создать и управлять таймером на вашем веб-приложении. Вы можете легко установить время для таймера, запустить его, поставить на паузу и сбросить.

## Цель проекта
Цель этого проекта - отработать навыки работы с setInterval в среде React и создать простое веб-приложение, демонстрирующее управление таймером.

## Как использовать
1. Установите зависимости с помощью `npm install` или `yarn install`.
2. Запустите приложение с помощью `npm start` или `yarn start`.
3. Откройте браузер и перейдите по адресу `http://localhost:3000`.
